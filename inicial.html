SI401

Explore o mundo da Web e se torne um desenvolvedor completo.

Com esse curso, a Unicamp oferece aos seus alunos um aprimoramento dos seus conhecimentos em aula.

Pratique com exercícios que acompanham o nível de aprendizado acumulado ao longo do curso.

Construa uma carreira com um diploma conquistado por meio de conhecimentos sólidos e úteis.

Ementa do curso

  • Elaboração de páginas web
  • Linguagens de marcação, Folhas de estilo e Javascript
  • Uso de linguagens de programação para geração dinâmica de páginas web
  • Uso de servidores web para armazenamento de sites
  • Atividades práticas de implementação de sistemas baseados nessas linguagens.

Objetivos

Colocar em perspectiva a programação para Web e capacitar o aluno a compreender as principais linguagens de marcação e estilo, compreender e desenvolver scripts para execução em navegadores e scripts para execução em servidores web.

Avaliação

A disciplina contará com um Projeto Final (em grupo), dividido em três entregas parciais ao longo do semestre, e uma série de exercícios práticos de fixação. No Projeto Final, as notas de cada aluno serão atribuídas conforme sua contribuição em cada etapa.

Competências que você vai desenvolver

  • Compreensão do funcionamento de aplicações web baseadas em arquitetura cliente-servidor.
  • Criação de páginas web estáticas baseadas em HTML e CSS.
  • Desenvolvimento de páginas web dinâmicas.
  • Desenvolvimento de aplicações web completas, com integração entre front-end e back-end.

Veja como essas competências serão desenvolvidas em aula

Informações técnicas

  • Carga horária total: 60 horas (15 semanas).
  • Frequência mínima: 75%.
  • Pré-requisitos: SI300
  • Exame: Obrigatório se 2,5 < M < 6,0
